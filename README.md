# Remy Splat Studio

本地 3D 高斯泼溅动画与视频制作器。它直接读取 MipMap 导出的二进制 PLY，用完整椭球投影、透明度合成、相机深度排序和球谐颜色渲染模型，并通过 GPU 生成粒子聚合动画。

## 直接使用

双击 `dist/RemySplatStudio.exe`。程序会启动一个本地服务，并以 Edge 独立应用窗口打开界面；制作期间请保留小型启动器窗口。

1. 点击“导入高斯 PLY”，选择 `gs.ply` 或 `ue/gs_full.ply`。
2. 导入后先显示完整结束帧，可拖动鼠标旋转视角、滚轮调整距离并检查清晰度。
3. 调整时长、爆炸强度、扫描波宽度和高斯尺寸，然后点击“播放动画”。
4. 选择分辨率、帧率和码率，点击“录制视频”。动画完成后，视频保存到浏览器下载目录。

## `gs.ply` 与 `gs_full.ply`

- `gs.ply`：包含完整位置、旋转、尺度、透明度和基础颜色（SH0）。文件较小，载入更快。
- `ue/gs_full.ply`：额外包含三阶球谐系数（SH3），可随观察方向还原高光和颜色变化，最高画质建议使用它。
- 这组模型约有 266 万个高斯；`gs_full.ply` 约 660 MB，载入和渲染会占用较多内存、显存及 CPU 排序时间。

渲染器不再抽样，也不会把高斯当普通圆点绘制。动画结束帧会恢复原始椭球尺度和颜色；“高斯尺寸”保持 `1.00` 时最接近原始模型。

## 画质建议

- 常规成片：1080p、30 FPS、25 Mbps。
- 高质量：1440p、30 FPS、50 Mbps。
- 4K：建议先短片测试。266 万高斯的 4K/60 FPS 对硬件和实时编码压力很高。
- 若界面 FPS 低于目标帧率，录制可能掉帧。优先降低输出分辨率或帧率，不要降低高斯数量。
- 清晰度还受原始扫描质量、镜头距离和屏幕缩放影响；高斯尺寸过大会显得发糊。

## 运行要求与隐私

- Windows 10/11、Microsoft Edge、支持 WebGL2 的显卡。
- 启动器会请求 Edge 使用 D3D11 高性能 GPU；最终显卡选择仍受 Windows“图形设置”和 NVIDIA/AMD 控制面板影响。
- 模型只在本机读取和渲染，不会上传网络。
- 视频优先输出 MP4/H.264；系统不支持时自动回退到高码率 WebM。

## 重新编译

在项目目录运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\build.ps1
```

构建使用 Windows 自带的 .NET Framework C# 编译器，将界面、渲染引擎和许可证嵌入单个 EXE。3DGS 核心基于 PlayCanvas Engine 2.19.7（MIT），许可证见 `web/PLAYCANVAS-LICENSE.txt`。
