# GaussianSplatterEffectsStudio

本地完整 3D 高斯泼溅特效与电影运镜工作台。程序直接读取 MipMap 导出的二进制 PLY，用完整椭球投影、透明合成、相机深度排序和球谐颜色渲染模型，再通过 GPU 生成动画并录制视频。

## 直接使用

双击 `dist/GaussianSplatterEffectsStudio.exe`。程序会启动本地服务并打开 Edge 独立应用窗口；制作期间请保留小型启动器窗口。

1. 导入 `gs.ply` 或 `ue/gs_full.ply`。
2. 选择高斯特效和电影运镜，调整强度、时长等参数。
3. 点击“播放动画”预览；时间轴可以拖动检查任意阶段。
4. 选择分辨率、帧率和码率，然后点击“渲染视频”。程序会打开独立预览窗口；完成后可直接播放并点击“导出视频”。

导入后默认显示清晰的完整结束帧。“高斯尺寸”保持 `1.00` 时最接近原始模型。

## 手动视角

- 左键拖动：球面自由环绕，可连续跨越顶部和底部，不再受俯仰角极限卡住。
- 右键或中键拖动：平移观察中心。
- 滚轮：靠近或远离。
- 双击画布或点击“复位手动视角”：恢复默认构图。
- 播放期间开始手动操作会停在当前时间和当前机位，不会与自动运镜互相抢夺控制。
- “显示三维轨迹”会绘制蓝色轨迹、绿色起点、红色终点和黄色当前机位；录制时自动隐藏。
- 鼠标控制的工作视角与拍摄轨迹完全分离。鼠标环绕、平移和缩放不会修改最终视频的机位。

## 高斯特效

- 爆散聚合：粒子从四周和上方聚合成模型。
- 扫描显形：径向扫描波逐层还原高斯。
- 粒子雨：粒子由上向下落位。
- 旋涡聚合：绕模型旋转并收束。
- 晶格拼装：由规则空间晶格拼接回原形。
- 尘埃升腾：尘埃从下方上升并聚合。
- 能量脉冲：模型保持可见，能量波从内部向外扫过。

所有生成类特效在动画结束时恢复原始位置、椭球尺度、透明度和颜色。

## 电影运镜

- 静止、360° 环绕、旋转靠近、平滑推进、平滑拉远
- 升降揭示、横移弧线、8 字环绕、低机位英雄镜头

这些预设对应常见的 locked、orbit/arc、dolly、crane/pedestal 和 truck 类镜头，并针对单个 3DGS 主体做了平滑缓入缓出。运镜幅度为 `1.00` 时使用标准幅度。

运镜可以进一步自定义：

- 旋转中心 X/Y/Z：以模型包围盒中心为原点，数值单位是模型半径。
- 轨道轴：水平 Y 轴、垂直 X/Z 轴或用方位角与仰角定义任意倾斜轴。
- 顺/逆时针、0.25–3 圈旋转、起始方位角和起始俯仰角。
- 起始距离、结束距离倍率、镜头 FOV 和整体运镜幅度。
- 轨迹形状包括完整圆圈、指定角度圆弧、螺旋、直线、升降和 8 字轨迹。
- 轨迹圆心和视角跟随点相互独立。例如将圆心 Y 提高、跟随点留在模型中心，即可得到在物体上方盘旋并持续向下看的镜头。
- 起始/结束滚转角（Roll）控制摄像机绕自身视线轴旋转。两个角度相同是固定倾斜，不同则在运镜过程中连续滚转，同时保持相机位置与跟随点不变。
- `360° 环绕`、`旋转靠近`和`横移弧线`使用所选轨道轴；其他镜头仍围绕指定中心构图。

## `gs.ply` 与 `gs_full.ply`

- `gs.ply`：完整位置、旋转、尺度、透明度和基础颜色（SH0），文件较小。
- `ue/gs_full.ply`：额外包含三阶球谐系数（SH3），能随视角还原颜色和高光变化，最高画质建议使用。
- 当前模型约 266 万个高斯；约 660 MB 的 `gs_full.ply` 会占用较多内存、显存和 CPU 排序时间。

## 画质建议

渲染质量提供“高质量、平衡、流畅预览”三档。修改渲染或输出参数后，状态会显示“尚未应用”；点击“应用当前渲染设置”后才会显示绿色的实际生效配置。开始录制前也会自动应用当前选择。

- 常规成片：1080p、30 FPS、25 Mbps。
- 高质量：1440p、30 FPS、50 Mbps。
- 4K：先做短片测试；266 万高斯的 4K/60 FPS 压力很高。
- FPS 低于目标帧率时，优先降低输出分辨率或帧率。
- 视频优先输出 MP4/H.264；系统不支持时回退到高码率 WebM。

## 运行与编译

- Windows 10/11、Microsoft Edge、支持 WebGL2 的显卡。
- 启动器请求 Edge 使用 D3D11 高性能 GPU；最终显卡选择仍受 Windows 和显卡控制面板影响。
- 模型只在本机读取和渲染，不会上传网络。

重新打包：

```powershell
powershell -ExecutionPolicy Bypass -File .\build.ps1
```

构建使用 Windows 自带的 .NET Framework C# 编译器。3DGS 核心基于 PlayCanvas Engine 2.19.7（MIT），许可证见 `web/PLAYCANVAS-LICENSE.txt`。
